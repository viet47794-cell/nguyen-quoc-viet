* Prompt đã sử dụng:

"Hãy đóng vai là một chuyên gia Frontend Web Developer và UI/UX Designer. Viết mã nguồn HTML, CSS và JavaScript để xây dựng một trang web Portfolio cá nhân cho tôi với các yêu cầu chi tiết sau:

1. Thông tin cá nhân (Nội dung):

Tên: Nguyễn Quốc Việt.

Trình độ học vấn: Sinh viên năm 2 ngành Công nghệ Thông tin và Thiết kế Đồ họa tại Đại học Gia Định (GDU).

Đặc điểm nổi bật: Kết hợp tư duy logic lập trình và thẩm mỹ thiết kế.

Năng lực/Sở thích: Nhiếp ảnh (Sử dụng máy Canon EOS 1200D), thao tác phản xạ nhịp độ cao (maimai DX - rating 11,500).

2. Cấu trúc UI & Bố cục (Layout):

Bám sát 60% cấu trúc của một Portfolio học thuật tối giản, bao gồm các section: Header (Sticky Nav), Hero (Giới thiệu & Số liệu thống kê), Projects (Dự án tiêu biểu), Skills (Kỹ năng) và Contact.

Thiết kế Responsive, hiển thị tốt trên cả Desktop và thiết bị di động (Mobile).

3. Tính năng chức năng (Functionality - 40% độ tùy biến):

Theme Switcher: Tích hợp nút chuyển đổi giao diện Sáng/Tối (Light/Dark mode) sử dụng JavaScript và CSS Variables.

Smooth Scrolling: Thanh điều hướng (Navigation bar) cố định ở trên cùng, khi click vào các link sẽ trượt mượt mà đến từng section tương ứng.

Interactive Skills Section: Khu vực "Kỹ năng" không hiển thị văn bản tĩnh mà phải dùng hiệu ứng Accordion (Click vào từng thẻ kỹ năng để mở rộng/thu gọn nội dung chi tiết bên trong).

4. Yêu cầu mã nguồn:

Viết code sạch (clean code), chuẩn Semantic HTML5.

CSS không sử dụng thư viện ngoài, tối ưu hóa class name.

File JavaScript tách biệt logic rõ ràng, có comment chú thích dễ hiểu."
